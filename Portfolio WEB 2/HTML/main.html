PROYECTO DE JAVA

Este proyecto de Java es un programa CRUD (Create, Read, Update, Delete) en una Base de Datos MySQL con una interfaz gráfica que facilita las acciones propias de una Base de Datos.

He utilizado IntelliJ IDEA como IDE de programación para la parte funcional, mientras que he usado la herramienta gráfica de NetBeans para la interfaz.

Para la Base de Datos he usado Xampp junto Apache y MySQL, además de usar un MySQL Connector en las librerías del proyecto, utilizando un JDBC Driver.
